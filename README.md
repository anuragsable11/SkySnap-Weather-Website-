# 🌤️ SkySnap - Weather Web App

A modern and minimal **weather application** built using Django that provides real-time weather information for any city worldwide.

---

## 🚀 Features

* 🔍 Search weather by city name
* 🌡️ Real-time temperature (°C)
* 🌤️ Weather condition & description
* 💧 Humidity & wind details
* 🕒 Local date & time (based on city)
* 🎨 Clean modern UI (glassmorphism style)
* 🌐 Dynamic weather icons

---

## 🛠️ Tech Stack

* **Backend:** Django (Python)
* **Frontend:** HTML, CSS
* **API:** Weather API (OpenWeather)

---



## ⚙️ Setup Instructions

### 1. Clone the repository

### 2. Create virtual environment

```bash
python -m venv venv
venv\Scripts\activate   # Windows
```
### 3. Install dependencies
```bash
pip install -r requirements.txt
``
### 4. Create `.env` file
Create a file named `.env` in the root directory and add:

```env
API_KEY=your_api_key_here
```

---

### 5. Run the server

```bash
python manage.py runserver
```

Open browser:

```
http://127.0.0.1:8000/
```

---

## 🔐 Environment Variables

| Variable | Description          |
| -------- | -------------------- |
| API_KEY  | Your weather API key |
